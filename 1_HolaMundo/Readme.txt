
El fichero holamundo.html contiene la estructura básica para empezar a trabajar con BootStrap.

El fichero holamundo2.html además de contener la estructura sirve para empezar a trabajar con contenedores.

https://getbootstrap.com/docs/4.3/layout/overview/#containers

	Los contenedores son los elementos más básicos para estructurar contenido en Bootstrap y son necesarios para usar el sistema por defecto del grid.
	
	Se pueden usar contenedores "responsive" y de ancho fijo o de ancho fluido
	Choose from a responsive, fixed-width container (meaning its max-width changes at each breakpoint) or fluid-width (meaning it’s 100% wide all the time).

While containers can be nested, most layouts do not require a nested container.
