Tarjetas

Las tarjetas son un container flexible y extensible. Incluye opciones para cabeceras y pies, y una amplia variedad de contenidos, colores y potentes opciones de visualización. Para más información puedes ver la documentación oficial.

Ver documentación oficial.
Carousel

El componente de carrusel nos permite crear galerías de imágenes y contenido que va pasando de manera automática, esto es muy común en webs para darle importancia a contenido actualizado o exclusivo. Existen varios modos de carrousel para más información puedes ver la documentación oficial.

Ver documentación oficial.
Layout

La rejilla (grid) de Bootstrap permite estructurar una web y colocar el contenido en la posición deseada con facilidad. Escala de manera apropiada hasta 12 columnas teniendo en cuenta el tamaño de la ventana.

Ver documentación oficial.

Footer

Autor: Eliseo Martín Mora 10/06/2019

El footer o pie de página en una web es utilizado para cerrar el contenido, en él podemos encontrar enlaces de interés, información del copyright, etc. Puedes ver ejemplos en esta web.


Vamos a crear un footer en esta web para ello tendremos que crear una nueva fila y ponerle el fondo gris. Posteriormente podemos añadir enlaces a redes sociales, enlaces dentro de nuestra web y un logo de empresa, por lo que haremos tres columnas dentro de esta nueva fila.
